# Changelog

## [v2.0.0](https://github.com/sam1064max/agentloop/releases/tag/v2.0.0) (2026-03-28)
### Features
- Experimentation Engine: full A/B testing lifecycle for agent configs
- Statistical Attribution: 6 hypothesis tests (Chi-Square, Fisher, Welch T, Mann-Whitney U, Bayesian, Bootstrap)
- Cohort Analytics: user segmentation (new/returning, enterprise/SMB, region, version)
- Recommendation Engine: 5 deterministic recommendation types with priority scoring
- Regression Detection: automatic incident creation (prompt/tool/workflow/agent regressions)
- Agent Version Intelligence: lineage trees, version diff, trend analysis
- KPI Impact Engine: business translation (savings, revenue, efficiency)
- SDK Layer: Python (agentloop-sdk) and TypeScript (@agentloop/sdk) client packages
- Advanced Dashboard: Experiments, Recommendations, Incidents, Version Explorer, Cohorts, Business Impact pages
- 500K session synthetic data generator with injected regressions and true improvements
